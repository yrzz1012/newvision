# 01 — 需求规格

## 产品定位
一个以**3D高斯泼溅（Gaussian Splatting）**为核心展示技术的空间浏览与分享平台。用户可浏览酒店、民宿、商铺、展览等3D空间，并支持注册登录、上传自己的空间作品。

## 目标用户
- **浏览者**：寻找民宿/酒店/展览空间的消费者，通过3D漫游提前感受空间氛围
- **创作者**：民宿房东、空间设计师、摄影师，上传3D扫描作品展示自己的空间

## 核心功能

### P0 (MVP必备)
| 功能 | 描述 | 参考来源 |
|------|------|----------|
| 用户注册/登录 | 邮箱注册、登录、登出 | 新片场 |
| 首页Hero轮播 | 精选作品大图轮播，毛玻璃叠加标题 | 新片场 |
| 分类浏览 | 精选/民宿/展览/商业空间，4个分类Tab | 美团民宿 |
| 作品详情页 | 3D Viewer + 作品信息面板 | 贝壳VR + 新片场 |
| 3D空间漫游 | 基于SuperSplat的3D高斯泼溅全景漫游 | 核心亮点 |
| 作品上传 | 登录用户上传.ply文件 + 填写基本信息 | 新片场 |
| 搜索 | 按标题/分类搜索作品 | 通用 |

### P1 (后续迭代)
| 功能 | 描述 |
|------|------|
| 作品收藏 | 收藏喜欢的空间作品 |
| 个人主页 | 展示用户上传的作品和收藏 |
| 评论互动 | 作品下方评论交流 |
| 分享 | 生成分享链接/二维码 |

## 页面清单
| 页面 | 路由 | 权限 |
|------|------|------|
| 首页 | `/` | 公开 |
| 发现页 | `/explore` | 公开 |
| 作品详情 | `/work/[id]` | 公开 |
| 登录 | `/auth/login` | 未登录 |
| 注册 | `/auth/register` | 未登录 |
| 上传 | `/upload` | 需登录 |
| 个人主页 | `/profile/[username]` | 公开 |
| 搜索 | `/search?q=` | 公开 |

## 非功能需求
- **性能**：首页LCP < 2.5s，3D Viewer加载 < 5s
- **响应式**：移动端/平板/桌面端均适配
- **安全**：Row Level Security 保护用户数据
- **可扩展**：文件存储后续可从Supabase迁移至Cloudflare R2
