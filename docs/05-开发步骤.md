## 阶段1：设计系统 & 布局框架

**目标**：Vision Pro主题生效，Navbar+Footer+首页骨架渲染

### 任务
1. Tailwind 主题配置（`tailwind.config.ts` 添加 Vision Pro 设计Token）
2. `globals.css` 添加 CSS 变量 + 毛玻璃工具类
3. `ThemeProvider` 客户端组件（强制暗色模式）
4. `Navbar` 组件（毛玻璃sticky，Logo + 导航链接 + 搜索框 + 头像/登录 + 上传按钮）
5. `Footer` 组件
6. `app/layout.tsx` 整合全局布局
7. `app/page.tsx` 首页骨架（Hero占位 + CategoryNav占位 + 卡片网格占位）

### 验证标准
- 浏览器看到完整的Navbar + Footer
- Navbar 有毛玻璃半透明效果（需滚动页面验证sticky）
- 页面无控制台错误

## 阶段2：数据库 & 认证

**目标**：用户可以注册、登录、登出

### 任务
1. Supabase CLI 或 Dashboard 执行数据库迁移（profiles, works, favorites表 + RLS + Storage buckets）
2. 安装 `@supabase/ssr` `@supabase/supabase-js`
3. `lib/supabase/client.ts` 浏览器端客户端
4. `lib/supabase/server.ts` 服务端客户端
5. `AuthProvider` Zustand store
6. 登录页 `/auth/login`
7. 注册页 `/auth/register`
8. 导航栏登录态联动
9. 中间件保护需登录路由

### 验证标准
- 完整流程：注册 → 收到Supabase确认邮件 → 登录 → 导航栏显示头像 → 登出
