# 03 — 设计规范（Vision Pro 主题）

## 设计哲学

参考 Apple Vision Pro 的 visionOS 设计语言：
- **空间感**：深度层级通过模糊+透明度营造
- **聚焦**：交互元素有明确的光晕聚焦反馈
- **克制**：少即是多，大面积留白
- **沉浸**：深色模式优先，减少视觉噪音

## 毛玻璃样式 (Frosted Glass)

### 导航栏玻璃
```css
.glass-nav {
  background: rgba(10, 10, 11, 0.72);
  backdrop-filter: blur(24px) saturate(180%);
  border-bottom: 1px solid rgba(255, 255, 255, 0.06);
}
```

### 卡片玻璃
```css
.glass-card {
  background: rgba(255, 255, 255, 0.04);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.06);
}
.glass-card:hover {
  background: rgba(255, 255, 255, 0.06);
  border-color: rgba(255, 255, 255, 0.12);
}
```

## 动效规范

| 场景 | 时长 | 缓动函数 |
|------|------|----------|
| hover缩放 | 300ms | `cubic-bezier(0.25,0.1,0.25,1)` |
| 页面过渡 | 400ms | `cubic-bezier(0.4,0,0.2,1)` |
| 弹窗出现 | 300ms | spring-like (scale 0.95→1) |
| 轮播切换 | 500ms | `cubic-bezier(0.4,0,0.2,1)` |
| 加载骨架屏 | 1500ms | shimmer动画 |

## 间距 & 布局

- 页面水平padding: `px-4` (16px) 移动端, `px-8` (32px) 桌面端
- 区块垂直间距: `py-16` ~ `py-24` (64-96px)
- 卡片间距: `gap-4` (16px) ~ `gap-6` (24px)
- 内容最大宽度: `max-w-7xl` (1280px) 居中
