## 3D技术方案

### SuperSplat Viewer 集成方式
使用 `@playcanvas/supersplat-viewer` npm包，**iframe自托管**模式：

```
1. npm install @playcanvas/supersplat-viewer
2. 提取Viewer的HTML/CSS/JS静态文件放入 public/viewer/
3. React组件通过 iframe 嵌入，配置URL参数：
   - content: 3D文件URL (.sog)
   - noui: 隐藏内置UI
   - aa: 抗锯齿
   - settings: 体验设置JSON路径
```

### 上传转换流程
```
用户上传 .ply
  → Supabase Storage (bucket: splat-uploads)
  → API Route 触发 splat-transform CLI
  → 生成 .sog 文件存入 bucket: splat-public
  → 更新 works.splat_file_url
  → 前端Viewer加载 .sog URL
```
